# ChattingApp

유저간에 메시지 전송을 하는 어플, Firebase로 저장하여 보여줌 

## 개발 환경

1. kotlin,Firebase
2. targetSdkVersion 32, minSdkVersion 23
3. jetpack (lifecycle, recyclerview)사용
4. Firebase 사용 
5. constraintlayout 기반의 UI구성


## 기본구현

1. Firebase를 통한 유저정보 및 메시지 저장 구현 
2. 회원가입 구현 
3. 로그인 인증 구현 
4. 로그인 후 메시지 송수신 구현
